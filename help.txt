üîß Stripe Setup Guide
To test Stripe payments, follow these steps:

1. Create a Stripe Account
Go to https://dashboard.stripe.com and sign up.

2. Get API Keys
From the Stripe Dashboard ‚Üí Developers ‚Üí API Keys:

env
STRIPE_SECRET_KEY="sk_test_..."
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY="pk_test_..."

3. Create a Product + Price
Go to Products ‚Üí Add Product:

Name: ContentCal Pro
Price: $5.00 / month (recurring)
Copy the Price ID (starts with price_)

env
STRIPE_PRO_MONTHLY_PRICE_ID="price_..."

4. Set Up Webhook (for local testing)
Install the Stripe CLI:

Bash
# Windows (using scoop)
scoop install stripe
# Or download from https://stripe.com/docs/stripe-cli

Then run:
Bash
stripe login
stripe listen --forward-to localhost:3000/api/stripe/webhook

It will output a webhook signing secret:

env
STRIPE_WEBHOOK_SECRET="whsec_..."
